scikit.pyの説明

概要
	過去のレースから馬の予測順位及び勝率を計算する。

用意するデータ(20171217現在)
	scikit.py 123行、129行、137行参照

条件
	scikit.py、basic_wins.py、horce_nameが同じフォルダに入っていること
	
使用方法
	①scikit.py 142行目のWhERE tkd.url = の後に結果を見たいレースのURLを記載
	②出力したいファイル名を決め、229行に記載。フォーマットはcsv形式
	③SVM後の処理を決める、福田式なら233行〜251行までのコメントアウトを用いる。
	　あすか式なら253行から257行のコメントアウトを使用する。
	④scikit.pyをコンパイル、実行

出力フォーマット
	・ファイルパスはscikit.pyがあるフォルダ
	・ファイル内は[horce_name_id:SVMで計算した予測順位:勝率]

懸案
	・出力後のファイルにて,が大量発生。→テキストエディタの一括置換で対応可能
